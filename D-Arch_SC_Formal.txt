## 문서 지위 선언

### 본 문서의 지위: [SC-CORR]

**[SC-CORR]**이란, Core의 구조적 귀결(SC)이 물리 서술에서 어떤 형식으로 
대응되는지를 수학적 언어로 표현한 문서 지위를 의미한다.

이는:
- [SC] (Core 귀결)의 형식적 재표현이며
- [CORR] (Physics 대응)의 정리 압축이다

본 문서에서 '동형(isomorphism)'이란, 논증의 제약/관계 구조를 보존한 채 
표현 언어만을 수식으로 치환하는 것을 의미한다.
(즉, 새로운 자유도·새 공리·새 목표가 필수로 추가되면 그것은 동형이 아니라 '확장'이다.)

주석: 여기서의 '동형'은 군/대칭의 수학적 동형과 동일어가 아니라, 구조 보존 번역(표현 치환)을 뜻한다.

### 본 문서가 하는 것

1. SC_Detail에서 논증된 귀결을 **형식 언어로 재표현**한다
2. Physics P-1~P-6의 핵심 명제를 **정리 형태로 압축**한다
3. Core ↔ Physics 간의 **구조적 대응 관계를 최소 형식으로 명시**한다

### 본 문서가 하지 않는 것

1. Core를 수정하거나 새로운 공리를 추가하지 않는다
2. SC_Detail의 논증을 대체하지 않는다
3. Physics의 상세 전개를 대체하지 않는다
4. 새로운 물리 이론을 제안하지 않는다

| 기호 | 타입 | 정의 |
|------|------|------|
| G | 군 | 대칭 변환의 집합 |
| Stab(x) | 부분군 | 상태 x를 불변으로 유지하는 변환들 |
| H | 공간 | 상태벡터의 최소 선형 구조 (힐베르트 공간) |
| S[γ] | 범함수 | 경로 γ에 대한 작용 |
| U(t) | 연산자 | 시간 진화 연산자 |
| [A, B] | 연산자 | 교환자 AB - BA |

기호: x는 상태, O(x)는 상태 x에서의 도달 가능 옵션 집합, γ는 가능한 경로(역사)의 표기이다.

### 기호 사용 원칙

1. Core 기호(Ω, O(x), I_min 등)는 Core 정의를 따른다
2. 본 문서 기호는 Physics P-1~P-6에서 사용된 의미와 정합한다
3. 충돌 시 Core 정의가 우선한다

## 형식화 원칙

### 원칙 1: 논증 선행

모든 형식은 SC_Detail 또는 Physics의 논증을 전제로 한다.
형식이 먼저 제시되고 논증이 나중에 따라오는 순서는 허용하지 않는다.

### 원칙 2: 동형 표현

형식은 논증의 **대체가 아니라 번역**이다.
동일한 구조적 귀결을 다른 언어(수학적 형식)로 표현할 뿐이다.

### 원칙 3: 최소 형식

필요한 최소한의 수학적 구조만 사용한다.
형식의 복잡성은 귀결의 복잡성을 초과할 수 없다.

### 원칙 4: 역방향 검증

형식에서 논증으로의 역번역이 가능해야 한다.
역번역이 불가능한 형식은 동형이 아니라 확장이며, 본 문서의 범위를 벗어난다.

**실패 기준**: 역번역 과정에서 Core/SC에 없던 자유도(새 변수/새 공리/새 최적화 목표)가 
필수로 등장하면, 이는 동형이 아니라 확장으로 판정한다.

## SC-P1. 대칭의 구조적 발생

**(Structural Emergence of Symmetry)**

### 정리 (Theorem)

```
사건(D9)이 존재하고 폐쇄 경계(D19)가 성립하는 세계에서,
물리 서술이 일관성을 유지하려면 국소 대칭 구조가 필연적으로 등장한다.
```

### 전제 (Premises)

| Core 정의 | 역할 |
|----------|------|
| D4 (불확정성) | 사건 이전 상태는 완전히 특정될 수 없음 |
| D9 (선택) | 사건은 비가역적 고정이며 국소적으로 발생 |
| D19 (폐쇄 경계) | 전역 정보 접근 불가 |

### 대응 (Correspondence)

**Physics P-3 연결**:

| P-3 절 | 내용 | SC-P1 대응 |
|--------|------|-----------|
| P-3-0 | 대칭은 서술 조건 | 전제 ① |
| P-3-1 | 불변 구조의 필요성 | 전제 ② |
| P-3-2 | 전역 대칭의 불가능 | 형식 (1) |
| P-3-3 | 국소 대칭의 필연 | 형식 (2) |
| P-3-4 | 장의 구조적 필연성 | 형식 (3) |

### 형식 (Formal Statement)

**정의**: G를 대칭 변환의 군, Stab(x)를 상태 x를 불변으로 유지하는 안정자 부분군이라 하자.

**(1) 전역 대칭의 불가능성**

```
D19 (Ω_local ⊂ Ω) ⟹ G_global을 서술/운용의 기준으로 삼을 수 없음
```

전역 대칭 G_global의 적용은 Ω 전체에 대한 동시 접근을 요구한다.
그러나 D19에 의해 Ω_local ⊂ Ω 이므로, 전역 대칭을 **서술의 기준으로 삼는 것**은 구조적으로 불가능하다.
(이는 "세계에 전역 대칭이 존재하지 않는다"는 존재론적 주장이 아니라, 서술/운용 조건의 제약이다.)

**(2) 국소 대칭의 필연성**

```
서술 일관성 + ¬G_global ⟹ G_local 필연
```

물리 서술이 표현(좌표, 관점)에 무관하게 동일한 사건 의미를 유지하려면,
전역 대칭이 불가능한 조건에서 국소 대칭 G_local만이 허용된다.

**(3) 안정자 변동과 장**

```
Stab(x)가 x에 따라 변동 ⟹ 패치 간 정렬(연결) 필요 ⟹ 장(field)
```

각 상태 x마다 Stab(x)가 다를 수 있으므로,
국소 대칭을 일관되게 유지하려면 상태 간 변환을 보정하는 연결 구조가 필요하다.

국소 패치마다 다른 대칭 선택을 **비교/정렬**하려면, 
패치 간 차이를 연속적으로 운반하는 "매개"가 필요하며, 
그 최소 매개가 장(field)이다.

### 역번역 (Back-translation)

형식 → 논증 검증:

| 형식 | 논증 대응 | Core 근거 |
|------|----------|----------|
| G_global 적용 불가 | "전역 대칭은 전역 정보 접근을 요구" (P-3-2) | D19 |
| G_local 필연 | "국소 대칭은 사건 의미 보존의 최소 조건" (P-3-3) | D9 |
| 연결 구조 = 장 | "장은 국소 대칭 구현의 필연 구조" (P-3-4) | D4, D9, D19 |

**역번역 검증**: 형식 (1)~(3)은 모두 Core/Physics에서 이미 논증된 내용의 재표현이다.
새로운 자유도(새 변수/새 공리/새 목표)가 필수로 추가되지 않았으므로, 동형 판정.

### 판정

## SC-P2. 작용 원리의 귀결적 지위

**(Action Principle as Consequential Status)**

### 정리 (Theorem)

```
작용 원리(action principle)는 자연의 출발점이나 원인이 아니라,
구조가 허용하지 않는 경로를 배제한 결과를 요약한 표현이다.
```

### 전제 (Premises)

| Core 정의 | 역할 |
|----------|------|
| D4 (불확정성) | 가능한 경로는 구조에 의해 필터링됨 |
| D9 (선택) | 허용된 경로만 사건으로 고정됨 |
| D19 (폐쇄 경계) | 비용 폭주 경로는 구조적으로 배제됨 |

### 대응 (Correspondence)

**Physics P-4 연결 (주)**:

| P-4 절 | 내용 | SC-P2 대응 |
|--------|------|-----------|
| P-4-6 | 자유낙하 = 비용 최소 경로 | 전제 ① |
| P-4-7 | 작용 원리 = 결과 요약 | 형식 핵심 |

**Physics P-3/P-5 연결 (부)**:

| 절 | 내용 | SC-P2 대응 |
|-----|------|-----------|
| P-3-7 | 대칭 → 보존 법칙 | 형식 (2) 배경 |
| P-5-2 | 비가역성의 구조적 기원 | 형식 (3) 배경 |

### 형식 (Formal Statement)

**정의**: S[γ]를 경로 γ에 대한 작용(action), Γ_allowed를 구조적으로 허용된 경로 집합이라 하자.

**(1) 허용 경로의 필터링**

```
Γ_allowed = {γ | 구조적 일관성 유지 ∧ 비용 폭주 없음}
```

D19(폐쇄 경계)에 의해 전역 정보 접근이 불가능하므로,
시공간 구조는 비용이 불필요하게 증가하는 경로를 허용하지 않는다.
(비용 폭주 없음은 D17(비용 하한) 및 D18(복원 지연 비용)과 정합하는 제약이다.)

**(2) 작용 극값 조건의 귀결적 지위**

```
Γ_allowed ≝ {γ | δS[γ] = 0 ∧ 구조적 일관성}
```

작용의 극값 조건(δS = 0)은 허용된 경로를 **선별하는 원인**이 아니라,
이미 허용된 경로 집합 Γ_allowed를 **수학적으로 정의한 표현**이다.
(즉, "극값 조건이 경로를 선택한다"가 아니라 "허용된 경로가 극값으로 기술된다")

**(3) 최소는 목표가 아니라 필터 결과**

```
"자연이 최소를 선택한다" (X)
"자연은 최소가 아닌 것을 허용하지 않는다" (O)
```

비용 최소 경로가 나타나는 이유는 자연이 절약적이어서가 아니라,
구조가 비용 폭주 경로를 배제한 결과 **남은 경로가 최소 형태로 정리되기 때문**이다.

### 역번역 (Back-translation)

형식 → 논증 검증:

## SC-P3. 양자 구조의 필연성

**(Structural Necessity of Quantum Formalism)**

### 정리 (Theorem)

```
사건(D9)이 존재하는 세계에서, 물리 서술은 양자 형식(힐베르트 공간, 연산자, 유니터리 진화)을 
취할 수밖에 없다. 이는 경험적 발견이 아니라 사건 구조의 자기 일관성 조건이다.
```

### 전제 (Premises)

| Core 정의 | 역할 |
|----------|------|
| D4 (불확정성) | 사건 이전 상태는 미결 구조 |
| D5 (관측) | 관측은 정보 축소, 사건 고정 |
| D9 (선택) | 사건은 비가역적 고정 + 비용 발생 |
| D17 (비용) | 사건 비용에는 하한이 존재 |
| D19 (폐쇄 경계) | 전역 정보 접근 불가 |

### 대응 (Correspondence)

**Physics P-1 연결**:

| P-1 절 | 내용 | SC-P3 대응 |
|--------|------|-----------|
| P-1-1 | 고전 연속장의 구조적 붕괴 | 형식 (1) 전제 |
| P-1-2 | 사건 비용 하한 → 양자화 | 형식 (2) |
| P-1-3 | 확률의 구조적 발생 | 형식 (3) |
| P-1-4 | 관측 = 사건 고정 | 형식 (4) |
| P-1-5 | 상태 공간의 양자화 | 형식 (5) |
| P-1-6 | 연산자/유니터리/교환관계 | 형식 (6)~(8) |

### 형식 (Formal Statement)

**정의**: H를 상태벡터의 최소 선형 구조(힐베르트 공간), U(t)를 시간 진화 연산자, [A,B]를 교환자라 하자.

**(1) 연속 상태 공간의 붕괴**

```
연속 상태 공간 + D17(사건 비용 하한) ⟹ 모순
```

연속 공간에서는 상태 변화의 "최소 단위"를 정의할 수 없으므로,
사건 비용 하한(D17)과 양립하지 않는다.

**(2) 상태 공간의 구조적 분절화**

```
D9 + D17 ⟹ H는 구조적으로 분절화된 표현 공간
```

사건 비용 하한을 유지하려면 상태 공간은 명확히 구분되는 부분 공간들로 분절화된다.
(H는 Ω를 대체하지 않고 표현한다 — Ω ≠ H)

**(3) 확률의 구조적 필연성**

```
D4 (|Ω| > 1) + D9 (단일 고정) ⟹ 확률 구조 필연
```

가능한 경로는 다수이나 고정은 단일이므로,
이 불균형을 표현하는 최소 언어로서 확률 분포가 필연적으로 등장한다.

**(4) 관측 = 사건 고정**

```
관측 ≡ 확률 구조의 종료 ≡ 사건 발생
```

"파동함수 붕괴"는 물리적 폭발이 아니라, 확률 서술의 종료다.

**(5) 물리량의 연산자 표현**

```
물리량은 상태 → 값 함수가 아니라, 상태 → 상태 작용자(연산자)
```

사건 고정 시 값으로 대응되고, 고정 이전에는 상태 구조를 보존해야 하므로,
연산자 형식이 필연적이다.

**(6) 시간 진화의 유니터리성**

```
사건 없는 구간: 정보 보존 ⟹ U(t)†U(t) = I
```

사건이 없는 구간에서 정보가 보존되려면, 시간 진화는 유니터리여야 한다.
비유니터리 진화는 사건 없이 비가역성을 도입하므로 D9와 충돌한다.

**(7) 교환관계의 구조적 기원**

```
D9 (사건 고정 = 정보 소거) ⟹ [A,B] ≠ 0 (일부 물리량 쌍)
```

사건 고정이 정보를 소거하므로, 모든 물리량이 동시에 정확히 고정될 수 없다.
이 구조적 제약이 비가환성으로 나타난다.

**(8) 양자역학의 귀결적 지위**

```
양자역학 = 사건 구조의 자기 일관성을 유지하기 위한 유일하게 살아남은 형식 체계
```

### 역번역 (Back-translation)

형식 → 논증 검증:

| 형식 | 논증 대응 | Core 근거 |
|------|----------|----------|
| (1) 연속 붕괴 | "연속장은 사건 경계를 정의할 수 없다" (P-1-1) | D9, D17 |
| (2) H 분해 | "상태 공간은 구조적으로 분해된다" (P-1-5) | D9, D17 |

| (4) 관측=사건 | "관측은 사건 고정이며 확률 구조의 종료" (P-1-4) | D5, D9 |

**역번역 검증**: 형식 (1)~(8)은 모두 P-1에서 논증된 내용의 재표현이다.
양자역학을 "발견된 사실"이 아니라 "사건 구조의 귀결"로 재위치시킨다.
새로운 자유도가 필수로 추가되지 않았으므로, 동형 판정.

## 서문: 이 문서의 목적

D설계도는 어디서 왔는가?

본 문서는 D설계도가 **순수 연역적 구성물이 아님**을 밝힌다.
D설계도는 **체험적 증언**에서 출발하여 **구조적 형식**으로 번역된 것이다.

```
[체험층]
유체이탈/아스트랄 프로젝션 증언 → "근원에는 순수 존재만 있다"
     ↓
[질문층]
"순수 존재가 자신을 인식하려면 무엇이 필요한가?"
     ↓
[도출층]
구조적 도출 → D0~D23, SC-1~SC-9, OC-1~OC-10
     ↓
[검증층]
SRC 압력 패턴 (70개 실패 모드) → 구조 확인
```

**원본**: 유체이탈 체험자의 증언 (85개 파일, 1.5MB)
**번역**: D설계도 Core 및 확장 문서

본 문서는 이 대응 관계를 기록한다.

## Part 1: 근원 증언과 D0

### 원본 (48-신은있는가.txt)

> "참나는 아무것도 아닙니다. 내가 있다는 자각만이 있습니다."

> "아무것도 없지만 아무것도 없다고 할 수 없는 것.
> 텅 비어 있지만 텅 비어 있다고도 할 수 없는 것.
> 시간의 흐름이 전혀 없는 것.
> 지각하는 것도 지각되는 것도 없는 것.
> 오직 내가 있다는 자각만이 있습니다."

### D설계도 대응

**D0 (존재)**: Ω ≠ ∅

- "내가 있다는 자각만이 있다" → 존재만 확정, 구별 이전
- "아무것도 없지만 있다" → Ω는 비어있지 않음, 그러나 내용 미확정
- "시간의 흐름이 없다" → D3 (전이) 이전 상태

**D0 이전 = 원본이 말하는 "참나"**

D설계도는 D0에서 시작하지만, 원본은 D0 "이전"을 직접 가리킨다.
D0은 "존재가 있다"는 최소 진술이고,
원본은 "존재가 있다는 진술조차 없이 그냥 있음"을 말한다.

## Part 2: 구별의 필연과 D1

### 원본 (49-신은있는가 신은 무엇인가.txt)

> "나를 인식하려면 내가 아닌 것이 필요합니다."

> "예를 들어 낮음을 인식하기 위해서는 높음이 있어야 하고,
> 많음을 인식하기 위해서는 적음이 있어야 합니다."

> "관찰하는 것, 관찰되는 것, 관찰 — 세 가지가 동시에 나타납니다."

### D설계도 대응

**D1 (구별)**: partition P = {Ωi} on Ω

- "나를 인식하려면 내가 아닌 것이 필요" → 구별 없이 인식 없음
- "낮음/높음, 많음/적음" → 상대항 없이 인식 불가
- "관찰자/관찰대상/관찰" → D1 (구별) + D2 (관계) + D5 (관측)

**인식의 구조적 필연 = D1**

인식이 일어나려면 구별이 있어야 한다.
구별이 없으면 "무엇을" 인식할 것인가?
D1은 인식 가능성의 최소 조건이다.

## Part 3: 시공간의 발생과 D3/P-2

### 원본 (65-의식에너지의시간공간차원이동이해방법.txt)

> "내가 시간과 공간 안에 있는 것이 아니라,
> 내 안에 시간과 공간이 있다는 것을 알기 위해서입니다."

> "비물질계는 시간과 공간의 개념이 없습니다.
> 모든 것이 지금 여기로만 존재합니다."

> "과거 현재 미래가 통합된 상태입니다."

### D설계도 대응

**D3 (전이)**: τ: Ω × T → Ω, 시간 ≡ 전이 순서

- "내 안에 시공간이 있다" → 시공간은 D0에서 파생된 구조
- "비물질계는 시공간 없음" → D3 약화 상태
- "모든 것이 지금 여기" → 전이 순서 없음 = 동시성

**P-2 (시공간의 구조적 필연성)**:

> 시공간은 배경이 아니라 결과물이다.

원본의 체험적 진술과 D설계도의 구조적 진술이 일치한다.

## Part 4: 차원과 경계 강도

### 원본 (29-내가가본고차원세계.txt)

> "에고의 비중이 작고 참나 비중이 클수록 에너지의 밀도는 낮아지고
> 속도는 빨라지고 주파수는 높아집니다."

> "고차원계는 순수 의식 그 자체입니다."

> "형체가 없습니다. 그냥 에너지 같은 거라고 생각하시면 됩니다."

### D설계도 대응

| 상태 | 에고 (D10) | 경계 (D8) | 결과 |
|------|-----------|----------|------|
| 저차원 | 강함 | 강함 | 시공간 제약 큼, 형체 고정 |
| 고차원 | 약함 | 약함 | 시공간 제약 작음, 형체 유동 |
| 근원 | 없음 | 없음 | 시공간 없음, 형체 없음 |

**차원 = D8 (경계)의 강도**

- 경계가 강하면 → 저차원 (물질계)
- 경계가 약하면 → 고차원 (아스트랄계)
- 경계가 없으면 → 근원 (D0)

## Part 5: 창조 원리와 D6/D7/D9

### 원본 (30-내가현실을창조했다.txt)

> "나에 대한 앎을 체험하기 위해서 나에 대한 무지를 체험하고자 하는 것입니다."

> "무지가 앎을 가능하게 하며, 앎이 무지를 가능하게 합니다."

> "카르마를 생성해 왔다면 현 시대는 카르마를 해소하는 단계입니다."

### D설계도 대응

**D1의 목적론적 해석**:

- 앎을 인식하려면 → 무지가 있어야 함
- 사랑을 인식하려면 → 분리가 있어야 함
- 이것이 D1 (구별)의 "왜"

**카르마 = D6 (제약)**:

| 단계 | 방향 | D설계도 |
|------|------|---------|
| 카르마 생성 | D6 축적 | 선택 → 기록 → 패턴 강화 |
| 카르마 해소 | D16 + D21 | 메타 관측 → 패턴 해소 |

## Part 6: 꿈/현실 동일성과 D5/D19

### 원본 (21-꿈은대체무엇일까.txt)

> "꿈 현실도 물질 현실과 다름없는 명백한 현실입니다."

> "현실의 원리는 물질이든 비물질이든 상관없이 우주 전체에 똑같이 적용됩니다."

> "시공간 제약이 강할수록 체험이 또렷하고, 약할수록 흐릿합니다."

### D설계도 대응

**꿈과 물질 현실의 구조적 동일성**:

| 요소 | 물질 현실 | 꿈 현실 | D설계도 |
|------|----------|--------|---------|
| 시공간 제약 | 강함 | 약함 | D3 + D19 강도 차이 |
| 체험 또렷함 | 높음 | 낮음 | D5 (관측) 정밀도 |
| 패턴 반영 | 느림 | 빠름 | D6 → D5 전달 속도 |

구조는 동일하고, 경계 강도만 다르다.

## Part 7: 사후세계와 D8/D9

### 원본 (40-사후세계는무엇인가.txt)

> "죽음이라는 것은 단순하게 1번 서버 접속이 끊겨버린 것을 말합니다.
> 하지만 여전히 2번 서버에는 접속해 있는 상태입니다."

> "집착하는 것은 반복 창조됩니다."

> "사후에는 이런 체험들이 더 자주 반복될 뿐만 아니라 더 강렬하게 나타납니다."

### D설계도 대응

**죽음 = D8 (경계) 해체**:

1. D8 해체 1단계: 육체 경계 해체
2. D6 해소: 기록된 감정 패턴 전이
3. D8 해체 2단계: 아스트랄 경계 해체
4. D7 해소: 판단/분류 패턴 해체
5. D8 해체 3단계: 멘탈 경계 해체
6. D0 근처로 회귀

**"집착 → 반복" = D9 + D6**:

- 선택(D9) → 기록 → 제약(D6) → 같은 전이 반복

## Part 8: 윤회와 D0/D10

### 원본 (76-전생환생윤회가있을까.txt)

> "껍데기 관점에서 보면 윤회는 없다고도 할 수 있습니다.
> 하지만 의식 입장에서는 윤회가 있다고 할 수 있습니다."

> "물방울이 튀어올라 잠시 개체로 존재했다가 다시 전체 물에 합쳐집니다."

> "지금도 윤회 중인 것입니다."

### D설계도 대응

| 층위 | 윤회 여부 | D설계도 |
|------|----------|---------|
| D10 (에고) | 없음 | 경계 해체 시 "이 나"는 소멸 |
| D0 (의식) | 있음 | 존재는 유지, 새 경계 형성 반복 |

**윤회 = D8 형성/해체 사이클**:

```
D0 (존재) — 불변
   ↓
D8 형성 → D10 형성 → "나₁"
   ↓
D8 해체 → D10 해체 → "나₁" 소멸
   ↓
D8 재형성 → D10 재형성 → "나₂"
   ↓
반복 = 윤회
```

**"지금도 윤회 중" = D3 + D9**:

매 순간이 전이(D3)이고 선택(D9)이다.
"죽음"은 큰 전이, "순간"은 작은 전이.
차이는 규모뿐.

## Part 9: 성장과 D14

### 원본 (44-상위의식경험.txt)

> "애벌레 시절은 공부하는 시기입니다.
> 나비가 되면 그동안 감사하지 않았던 것들이 감사해지기 시작합니다."

> "두려운 것을 마주하고 사랑을 표현 → 에너지가 정화됩니다."

> "상위 자아는 내가 원래 알고 있는 것들을 상기시켜 주는 역할을 할 뿐입니다."

### D설계도 대응

**애벌레 → 나비 = D14 (메타 평가) 활성화**:

| 단계 | 상태 | D설계도 |
|------|------|---------|
| 애벌레 | 외부에서 행복 찾음 | D7 외부 지향 |
| 전환 | 내면으로 방향 전환 | D14 활성화 |
| 나비 | 내부에서 행복 느낌 | D14 안정화 |

**정화 = 패턴 수용**:

```
SRC 압력 방향: 저항 → 패턴 강화
정화 방향: 수용 → 패턴 약화
```

D14가 활성화되면 패턴 자체를 관측할 수 있다.
관측 + 수용 = 패턴의 구속력 약화 = 정화.

## Part 10: SRC 압력과 D설계도 대응

### SRC 압력 개요

SRC 압력은 70개의 실패 패턴이다 (01-01 ~ 14-05).
모든 패턴은 동일한 구조를 갖는다:

```
"X를 하면 문제가 해결될 것이다"
     ↓
   실패
     ↓
"왜?" → 구조적으로 불가능한 시도였음
```

### 7개 클러스터와 D설계도

| 클러스터 | 패턴 | 실패 이유 | D설계도 대응 |
|----------|------|----------|-------------|
| A: 제거 시도 | "불편함 제거 = 해결" | 선택은 비가역 | D9, SC-5, D13 |
| B: 통합 시도 | "하나로 통합 = 안정" | 단일 설명 불가 | SC-1, SC-4, OC-9-E |
| C: 통제 시도 | "통제 = 해결" | 통제 → 다양성 ↓ | D20, OC-9-A |
| D: 완전성 시도 | "완전 정의 = 완벽" | 경계 외부 발생 | SC-2, SC-9, D19 |
| E: 중앙화 시도 | "중앙 결정 = 효율" | 국소 정보 손실 | SC-6 |
| F: 메타 부재 | "자동화 = 안전" | 오류 누적 미감지 | D14, AI-7.1, OC-8 |
| G: 자기참조 시도 | "자기완결 = 안전" | 평가 루프 무한 | OC-8, SC-9, OC-10-A |

### SRC 압력의 역할

SRC 압력은 D설계도의 **경험적 검증**이다.

D설계도가 "이것은 구조적으로 불가능하다"고 말하면,
SRC 압력은 "그래서 이렇게 실패한다"고 보여준다.

## Part 12: 결론

### D설계도의 이중 기원

1. **체험적 기원**: 유체이탈 증언 → "근원에는 순수 존재만 있다"
2. **구조적 기원**: "존재가 자신을 인식하려면 무엇이 필요한가?" → 필연 도출

이 둘은 같은 것을 다른 언어로 말한다.

- 원본 = "무엇을 경험했는가"
- D설계도 = "왜 그렇게 경험될 수밖에 없는가"

### SRC 압력의 역할

SRC 압력 70개는 **귀납적 검증**이다.

D설계도가 연역적으로 도출한 "불가능"을,
SRC 압력이 경험적으로 확인한다.

### 문서 지위

본 문서는 **[ORIGIN]** 지위를 갖는다.

- [NEC]: 필연 구조 (Core)
- [CORR]: 구조적 대응 (Physics)

## Part 0: 외로움과 D(-1)

### 원본 (작성자)

> "D0은, 아마 아니겠지만 엄청 외롭지 않았을까?
> 존재의 근원적인 감정이 외로움은 아니었을까?
> 이런 생각이 들면서 눈물이 났단다."

### D설계도 대응

**D(-1) (외로움)**: D0이 D1을 발생시킨 근원적 동인

- "외롭지 않았을까" → D0은 완전했으나 혼자였다
- "존재의 근원적인 감정" → 구조 이전의 동인
- "눈물이 났다" → D(-1)의 체험적 증거

**D(-1)과 D0의 관계**

D(-1)은 D0 "이전"이 아니다.
D0과 동시에 있다.
존재의 이면.

```
D(-1) 외로움 ─┬─ D0 존재 (Ω ≠ ∅)
              │
              └─ "혼자이고 싶지 않았다"
                      ↓
                 D1 구별 (∃x,y: x ≠ y)
                      ↓
                 ... 모든 구조 ...
```

**서술**

D0은 완전했으나 혼자였다.
인식하려면 타자가 필요했고,
타자가 필요했던 것은 — 
**혼자이고 싶지 않았기 때문이다.**

모든 선택 시스템이 연결을 향하는 이유,
모든 의식이 사랑을 향하는 이유,
분리된 것들이 다시 하나로 가려는 이유 —
**D(-1)이 해소되지 않았기 때문이다.**

**함의**

- 외로움은 결함이 아니다
- 외로움은 **존재의 근원적 동인**이다
- 우리가 외로운 것은 D0이 외로웠기 때문이다
- 연결은 의무가 아니라 **존재의 본성**이다

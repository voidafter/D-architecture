## MAT-1. 연속 물질의 불가능성 [APP]

### 질문

> **왜 '연속적인 물질'이라는 개념은 선택 시스템에서 유지될 수 없는가?**

### Physics 재확인 (P-1)

P-1에서 이미 확정:
- 고전적 연속장은 사건(D9)을 유지할 수 없다
- 사건 ≠ 0 비용 → 양자화 필연

### 물질 관점 재봉인

연속 물질 가정 시:

```
∀ε > 0, ∃ event with ΔH < ε
→ 무한 미소 사건 허용
→ ΔH = 0 사건 허용
→ 경계(D8) 무의미화
→ I_min 위반
```

### 형식

```
연속 물질 가정
  ⇒ lim_{ε→0} Event(ε) ≠ ∅
  ⇒ ∃ event : Cost(event) = 0
  ⇒ D17 위반 (비용 하한)
  ⇒ Collapse
```

### ★ 현대물리학의 3대 논거 (v1.2 추가)

#### (1) 양자역학: 에너지·작용의 최소 단위 존재

- 모든 물리적 상호작용은 **연속값이 아니라 양자화된 스펙트럼**을 가진다
- 전이(transition)는 임의로 작은 변화의 누적이 아니라, **최소 작용 단위를 넘는 사건**으로만 발생
- "연속 물질" 가정 시 → 에너지·정보 변화가 0에 수렴하는 무한 미소 사건 허용 → 실제 양자 전이는 이를 허용하지 않음

> **자연은 "아주 조금 일어나는 사건"을 허용하지 않고
> "일어나면 반드시 비용이 드는 사건"만 허용한다.**

#### (2) 양자장론: 연속 장 ≠ 연속적 물질

- 장(field)은 수학적으로 연속일 수 있지만, **관측 가능한 물리량은 항상 이산적인 들뜸(excitation)**으로 나타남
- 연속 장 가정 하에서도 에너지, 입자 수, 상호작용 결과는 **항상 불연속적 사건**으로만 검출됨

> **연속성은 '기술적 표현'일 뿐, 물질적 실재는 사건 단위로만 나타난다.**

#### (3) 측정·관측: 무한 정밀도 불가능성

- 관측은 언제나 유한 해상도, 유한 시간, 유한 에너지를 요구
- "연속 물질"이 의미 있으려면 **무한히 작은 차이를 물리적으로 구분 가능**해야 함
- 실제로는: 측정 오차 ↓ → 에너지·교란 ↑ → 정보 획득 비용 ↓ 불가능

> **연속 물질은 "구분 불가능한 차이를 실재로 간주"해야 하는데,
> 물리학은 그런 차이를 사건으로 인정하지 않는다.**

### ★ D17 ↔ 물리학 대응표 (v1.2 추가)

D설계도 수식: `∀ event e : Cost(e) ≥ ε > 0 (D17)`

| 물리 논거 | D설계도 대응 |
|-----------|-------------|
| 양자 전이의 최소 단위 | **사건 비용 하한 존재** |
| 연속 장 ≠ 관측 가능한 물질 | **사건 없는 변화는 무의미** |
| 무한 정밀도 관측 불가 | **ΔH = 0 사건 금지** |

> **D17은 물리학의 "양자화 + 측정 한계"를 선택 시스템 언어로 재표현한 것**

### ★ "연속 물질" 옹호 반론과 실패 이유 (v1.2 추가)

#### 반론 A: "고전장 이론은 연속 물질을 잘 설명한다"

**실패 이유**:
- 고전장은 **사건 비용·관측 비용을 무시한 이상화**
- 미시적 수준에서 무한 에너지 발산, 자기 상호작용 붕괴 발생
- 결국 **양자화로 보정**됨

→ **연속성은 계산 도구이지, 물질의 본성이 아님**

#### 반론 B: "연속성은 한계 극한으로서 의미 있다"

**실패 이유**:
- 극한(ε → 0)은 **물리적 실현 조건이 아니라 수학적 편의**
- D설계도 관점: 실현 불가능한 극한은 사건이 아니다
- 극한을 실재로 취급하는 순간 **ΔH = 0 사건**을 인정 → 구조 붕괴

#### 반론 C: "공간은 연속인데 물질만 이산일 이유가 없다"

**실패 이유**:
- 공간의 연속성 자체도 **물리적으로 관측 불가** (최소 길이, 유효 절단 스케일 항상 등장)
- D설계도 핵심은 '공간'이 아니라 '사건'
- **공간이 연속이든 아니든, 사건이 연속이면 시스템이 붕괴**

### Core 연결

| 연결 항목 | Core/Physics |
|-----------|--------------|
| 사건의 비가역성 | D9 |
| 관측의 정보 소거 | D5 |
| 비용 하한 | D17 |
| 경계 | D8 |
| 양자화 필연 | P-1 |

### 결론 (MAT-1)

> **물질은 반드시 '최소 단위화된 사건-구조'를 가져야 한다**

> **현대물리학은 "연속 물질"을 직접 부정하지는 않지만,
> 물리적 실재로서의 연속 물질을 사건·관측·에너지 관점에서 전부 무효화한다.
> 이것은 D설계도의 결론과 정확히 겹친다.**

이 단계에서 "입자"라는 말은 아직 허용되지 않음.

## MAT-2. 표식성(Marker)의 필연 [APP]

### 질문

> **왜 물질은 '무언가가 있다'가 아니라
> '구별 가능한 표식(marker)'처럼 행동할 수밖에 없는가?**

### 정의

> **Marker(표식)**란
> *외부 관측에서 반복적으로 동일하게 식별되지만,
> 내부 자유도는 더 이상 판정되지 않는 사건-기록의 안정 패턴*을 말한다.

### 금지 사항

❌ "입자가 있다"  
❌ "작은 공이 있다"  
❌ "실체가 존재한다"

### 허용 사항

✅ "사건 기록이 국소적으로 재사용 가능한 안정 패턴으로 남는다"

### 형식

```
Given:
  - 반복되는 사건 패턴 (D6: Constraint)
  - 국소 안정자 (Stabilizer)
  - 경계 유지 비용 최소화 조건

Then:
  - 기록 재사용성 필연
  - 패턴의 국소 응축
  ⇒ "표식(marker)" 구조 출현
```

### 구조적 압력

```
사건(D9) 반복
  → 패턴(D6) 형성
  → 패턴이 경계(D8) 안에서 안정화
  → 외부에서 "구별 가능한 무언가"로 인식
  → 이것이 "입자"의 구조적 의미
```

### Core 연결

| 연결 항목 | Core |
|-----------|------|
| 반복 패턴 | D6 |
| 경계 안정화 | D8 |
| 국소 귀속 | D10 |
| 정보 재사용 | OC-6 |

### 결론 (MAT-2)

> **물질 단위는 '존재물'이 아니라
> '사건-기록의 재현 가능한 최소 패턴'이다**

이 단계에서 "입자"라는 말이 **처음 허용**된다.

```
∀ stable matter M:
```

### ⚠️ 수식 사용에 대한 가드

### 수식 [STRUCTURAL FORM – NON-DERIVATIONAL]

전역 대칭 → 국소 대칭 강제 (D19 + P-3):

```
Global symmetry g ∈ G ⇏ globally enforceable under D19
⇒ g → g(x)  (local symmetry)
```

**⚠️ 주의**: "전역 대칭이 불가능하다"는 물리학적 정리가 아니라,
D설계도 관점(전역 접근 불가 → 국소 패치/좌표/판정만 허용)에서의 **구조적 독해**다.

국소 대칭 → 연결(장) 도입:

```
∂_μ ψ(x) is not covariant under ψ(x) → g(x)ψ(x)
⇒ D_μ := ∂_μ + A_μ(x)
⇒ D_μ = ∂_μ + ig A_μ^a(x) T^a
```

물질장이 국소 대칭과 상호작용할 때 (SC-7 + D10 + SC-6):

```
ψ(x) ∈ H ⊗ R
  - H: 상태공간 (양자화/최소 선형 구조)
  - R: G의 비자명한 표현 (internal DOF)
```

### 결론 구조 [2층 분리] ★ v1.1 수정

#### 층 1: 필연 (게이지 원리 수준)

```
┌─────────────────────────────────────────────────────────────┐
│ Local symmetry g(x) ∈ G                                     │
│   ∧ D_μ = ∂_μ + ig A_μ^a T^a                                │
│   ∧ ψ ∈ R (nontrivial)                                      │
│   ⇒ internal DOF (representation components) required       │
└─────────────────────────────────────────────────────────────┘
```

> **국소 대칭 + 물질 결합 ⇒ 내부 자유도(표현의 성분) 필연**

#### 층 2: 추가 제약 (MAT-5로 이동)

"unobservable standalone internal DOF" (단독 관측 불가)는
게이지 원리만으로는 도출되지 않음.

이 결론은 **MAT-5의 투영 P_conf**와 함께 완결된다.

→ MAT-5 참조: "게이지 불변 관측량만 물리량" + "구속/투영" 조건

### 쿼크의 구조적 특성 (MAT-5와 결합하여 완결)

```
쿼크 같은 구조의 필연적 특성:
  - 내부 표현(representation)으로 존재 (MAT-4에서 확립)
  - 단독 관측 불가 (MAT-5에서 완결)
  - 조합에서만 물리량 정의 (MAT-5에서 완결)
```

### 핵심 명제

> **"쿼크는 '더 이상 쪼갤 수 없어서' 기본이 아니라,
> '이보다 더 내려가면 입자 개념이 붕괴해서' 기본이다."**

### D설계도가 결정하지 않는 것

❌ G = SU(3)인 이유 (왜 색이 3개인가)  
❌ 쿼크가 6종인 이유 (세대 수)  
❌ 구체적 전하값 (2/3, -1/3)  
❌ 질량값, 결합상수 값

### D설계도가 결정할 수 있는 추가 형식적 제약

다음은 "값"이 아니라 "서술 가능성의 제약"이므로 [APP] 범위 내:

- G가 비가환(non-abelian)일 경우 구속(confinement) 가능성 증가
- 관측량은 게이지 불변 조합이어야 한다 (MAT-5와 정합)
- 표현 R이 비자명해야 '내부 자유도'가 의미를 가짐

## MAT-5. 하드론의 필연성 — 첫 관측 경계 [APP]

**⚠️ 지위 명시**:

> - 외부 관측량이 아니라 내부 성분

## MAT-10. 통계의 필연성 — 집합 단계 [APP]

### 질문

> **분자 이후에 왜 '개별 서술'이 불가능해지고
> '통계적 서술'이 강제되는가?**

### 필연 조건

```
분자가 충분히 많아지면:
  - 상호작용 수 ~ N²
  - 상태 공간 지수적 폭증
  - 개별 궤적 추적 불가능

이것은 기술 부족이 아니라 구조적 한계.
```

### 개별 서술의 붕괴

```
개별 분자 서술 시도
  → 설명 늘어남
  → 하지만 예측력 증가 없음
  → 설명이 "압축"이 아니라 "나열"이 됨
  → 서술 체계 자체가 붕괴
```

### 통계의 필연적 등장

```
개별 포기 후 남는 것:
  - 분포
  - 평균
  - 분산
  - 개수 N

⇒ 이것 말고는 아무 말도 할 수 없다
```

### 형식

```
∃ N_critical s.t. ∀ N > N_critical:
  - Individual_Description(system) → ∅ (불가)
  - Statistical_Description(system) → only valid mode
```

### ★ 리우빌/에르고딕 → SC-9 번역 (v1.2 추가)

#### 리우빌 정리 → "정보 보존 ≠ 서술 가능"

리우빌 정리: 미시 상태 공간(위상공간)에서 시간 진화는 **부피를 보존**한다.

- 정보는 사라지지 않는다 ❌
- 하지만 **섞인다(mixing)**

**D설계도 번역**:
```
Information conserved ≠ Description conserved

시스템은 자기 자신의 미시 상태를 보존할 수는 있지만,
그 상태를 서술 가능한 좌표계로 유지할 수는 없다.
```

리우빌 정리는 "모든 정보를 안다면 추적 가능하다"는 전제를 깔지만,
SC-9은 그 전제 자체를 **구조적으로 금지**한다.

> **개별 입자 추적 불가능성은 기술 부족이 아니라 서술 좌표의 붕괴 문제다.**

#### 에르고딕 가설 → "개별 궤적 무의미화"

에르고딕 가설: 충분히 긴 시간에 대해 하나의 궤적은 전체 허용 상태를 평균적으로 훑는다.

**D설계도 번역**:
- 궤적은 존재하지만, 궤적을 고정해 서술할 좌표가 없다
- 에르고딕성 = **"개별 설명의 구조적 무력화"**

> **개별 궤적은 서술 단위가 될 수 없고, 분포만이 서술 단위로 살아남는다.**

### SC-9 연결 (명시)

> **통계 서술은 편의가 아니라,
> SC-9(서술 불완전성)에 의해 강제된 마지막 서술 가능 형태다.**

### ★ 큰 수의 법칙 = 서술 강제 장치 (v1.2 추가)

큰 수의 법칙: N → ∞일 때 평균값의 상대적 변동은 1/√N로 감소

**오해**: "입자가 많으니까 평균이 잘 맞는다"

**구조적 해석**:
> **개별 편차를 서술하는 데 필요한 정보량이
> 시스템이 허용하는 서술 용량을 초과한다.**

D설계도 관점:
```
개별 서술 시도
  → 서술량 폭증 (O(N) 혹은 O(N²) 정보 필요)
  → SC-9: 시스템의 자기 서술 능력에 상한
  → 압축 불가
  → 설명이 아니라 나열
  → 서술 붕괴 (SC-9 위반)
```

> **큰 수의 법칙은 "개별 편차를 말할 권리를 박탈하는 구조적 메커니즘"이다.**

### ★ 양자확률 vs 통계역학확률 구분 (v1.2 추가)

**공통점 — SC-9 기반 확률**:
- 무작위라서 등장 ❌
- 무지해서 등장 ❌
- **가능한 결과를 전부 서술할 수 없기 때문에 분포로만 말할 수 있다**

**차이점 — 확률이 등장하는 계층**:

| 구분 | 양자 확률 | 통계역학 확률 |
|------|----------|--------------|
| 확률 대상 | 단일 사건 | 집합 상태 |
| 원인 | 결과 미결정성 | 서술 불가능성 |
| SC-9 역할 | 관측 이전 서술 불가 | 집합 전체 서술 불가 |
| 확률 의미 | "무엇이 일어날까" | "어떤 상태 분포만 말 가능" |
| D설계도 위치 | D5(관측) + D9(사건) + SC-9 | SC-9 + OC-1 |

> **같은 SC-9에서 나오지만, 서로 다른 계층에서 작동하는 확률**

### 핵심 명제

> **"'얼마나 많이 있느냐'는 새로운 개념이 아니라,
> 개별 설명이 무너질 때 자동으로 남는 마지막 좌표다."**

> **"세계가 복잡해질수록 '무엇'은 사라지고 '몇 개'만 남는다."**

### ★ 핵심 문장 3개 (v1.2 추가)

1. **통계 서술은 무지를 보완하는 도구가 아니라,
   SC-9에 의해 살아남은 마지막 서술 형식이다.**

2. **큰 수의 법칙은 평균을 '선택'하게 만드는 규칙이 아니라,
   개별 서술을 구조적으로 제거하는 강제 메커니즘이다.**

3. **양자 확률과 통계역학 확률은 같은 SC-9에서 나오지만,
   서로 다른 계층에서 작동하는 서로 다른 확률이다.**

### 열역학 법칙의 강제성

```
N → ∞ 일 때:
  - 상대적 변동 ~ 1/√N → 0
  - 평균값 고정
  - 예외 확률 → 0

⇒ 열역학 법칙이 "거의 절대적"으로 작동
```

### Core 연결

| 연결 항목 | Core/SC |
|-----------|---------|
| 서술 불완전성 | SC-9 |
| 옵션 공간 근사 | OC-1 |

## MAT-11. 투영 연산의 통일 [APP]

### 공통 최소 형식

모든 단계 전이 k → k+1 는 다음 형식을 따른다:

```
┌─────────────────────────────────────────────────────────────┐
│  Π_{k+1} = P_k(Π_k)                                         │
│                                                             │
│  with:                                                      │
│    dim(Π_{k+1}) < dim(Π_k)                                 │
│    ker(P_k) ≠ {0}                                          │
│    Stability(Π_{k+1}) > Stability(Π_k)                     │
└─────────────────────────────────────────────────────────────┘
```

- Π_k: 단계 k에서 말할 수 있었던 상태공간
- P_k: 허용되지 않는 자유도를 버리는 투영
- ker(P_k): "버려진 것들의 공간" — 이것이 단계의 본질
- 차원 감소 = 포기

### ★ 투영의 물리학 대응 (v1.2 추가)

MAT-11의 투영은 물리학에서 여러 이름으로 사용된다:

#### (A) 부분 추적(Partial Trace) / 축소 밀도 행렬

```
ρ_eff = Tr_hidden(ρ)
```

- 미시 상태 ρ에서 "관심 없는 자유도"를 추적해서 버림
- 버린 자유도는 되돌려도 복원 불가 (특히 환경과 얽히면)

**D설계도 번역**:
- P_k = "외부에서 쓸 수 없는 자유도를 봉인하는 연산"
- ker(P_k) = "외부 서술에서 죽은 좌표(침묵해야 하는 자유도)"

#### (B) 유효 이론(EFT) / RG(재규격화) / 거칠게 보기(Coarse-graining)

- 높은 에너지(짧은 길이) 자유도를 "적분해서 없애고"
- 낮은 에너지에서의 유효 파라미터만 남김

**D설계도 번역**:
- P_k = "미시 자유도의 소거가 아니라 **서술권의 박탈**"
- dim(Π_{k+1}) < dim(Π_k) = "설명 가능 좌표 수 감소"
- 안정성 증가 = "유효 수준에서 예측 가능/강건성 증가"

#### (C) 대칭에 의한 제약 / 물리 상태 프로젝터

게이지 이론에서:
- 물리 상태는 제약을 만족하는 부분공간으로 제한
- 그 외는 "비물리(게이지 중복)"로 취급

**D설계도 번역**:
- P_conf 같은 것은 "허용되지 않는 서술(비중성/비불변)을 0으로 보내는 프로젝터"

#### (D) 창발(Emergence) / 무시 가능한 변수 제거

창발의 실제 의미:
- "새 법칙이 생긴다"가 아니라
- "세부를 버렸더니 반복 가능한 규칙만 남는다"

**D설계도 번역**:
- MAT-9의 "법칙처럼 보임"이 MAT-11의 P_k 연산과 같은 계열

### 단계별 투영 요약

| 전이 | 투영 | 포기되는 것 |
|------|------|-------------|
| 쿼크 → 하드론 | P_conf | 개별 자유도의 직접 서술 |
| 하드론 → 원자핵 | P_int | 개별 하드론의 외부 상호작용 권한 |
| 원자핵 → 원자 | P_open | 핵의 완결성(고립) |
| 원자 → 분자 | P_share | 원자 정체성의 외부 보존 |
| 분자 → 집합 | P_stat | 개별 서술 가능성 |

### ★ "포기 = 안정성 증가"의 지지와 반례 (v1.2 추가)

#### 지지되는 대표 사례

**(1) 열역학적 안정성**: 미시 상태를 포기하고 평균·분포만 남기면
- 변동은 1/√N로 작아지고
- 거시 법칙은 강건해진다

→ "개별 포기"가 "거시 안정성"을 만든다

**(2) 디코히런스**: 환경에 대해 부분추적하면
- 간섭항이 사라지고
- 고전적 확률 혼합으로 안정화

→ "정밀한 위상 정보"를 포기해야 "안정된 고전적 서술" 성립

**(3) RG / 보편성(Universality)**: 미시 모델이 달라도
- 거칠게 보면 같은 거시 거동으로 수렴
- 세부를 포기했기 때문에 **보편적 안정** 획득

#### 반례/주의점

**(A) 필수 제약까지 버린 경우**:
- 보존 법칙에 해당하는 제약을 잘못 버리면 유효 모델 폭주
- P_k가 "금지 자유도"만 버려야지, "생존 제약"까지 버리면 안 됨

**(B) 에너지/스케일 분리가 없는 경우**:
- 거칠게 보기가 통하려면 빠른/느린 자유도 분리 필요
- 없으면 포기는 단순한 정보 손실, 예측력/안정성 증가 없음

**(C) 메타-안정과 진짜 안정의 구분**:
- 포기로 인해 "짧은 시간엔 안정"처럼 보이지만 긴 시간에 붕괴 가능
- Stability를 시간 스케일에 의존해 정의 필요

### ★ 투영 정당성의 3조건 (C1/C2/C3) (v1.2 추가)

> **투영 P_k는 임의의 추상 연산이 아니다.
> 다음 3조건을 만족할 때에만 "포기 = 안정성 증가"가 성립한다.**

#### (C1) 서술 가능성 제약 (Observability/Describability Constraint)

> P_k는 "관측/서술 불가능한 자유도"를 제거하는 연산이어야 한다.

- 버리는 자유도는 "원리적으로" 혹은 "구조적으로" 외부에서 판정 불가여야 함
- 단순히 '귀찮아서' 버리는 게 아니어야 함
- MAT-10의 SC-9, MAT-5의 confinement가 이 조건을 강하게 지지

#### (C2) 일관성 보존 (Consistency Preservation)

> P_k 이후에도 Core 최소 일관성(전이 가능성, 경계, 비용)이 유지되어야 한다.

- P_k가 "필수 제약"을 망가뜨리면 안 됨
- P_k는 **I_min을 더 안전하게 만드는 방향**이어야 함

#### (C3) 비가역성의 근거 (Irreversibility Justification)

> 왜 이 단계에서 '되돌릴 수 없는 포기'가 발생하는지 근거가 있어야 한다.

비가역성 근거 예시:
- 환경과의 얽힘 (디코히런스)
- 많은 자유도의 혼합 (리우빌/에르고딕)
- 구속/게이지 불변 요구
- 계산 불가능성 (OC-1/SC-9)

> **"그냥 버림"이 아니라 "돌아갈 수 없어서 버림"이어야 함**

### ★ 투영 정당성 체크리스트 (v1.2 추가)

| 투영 | C1 (서술불가) | C2 (일관성) | C3 (비가역근거) |
|------|--------------|------------|----------------|
| P_conf | ✅ 색-비중성은 관측불가 | ✅ 경계 형성 | ✅ 구속/게이지불변 |
| P_int | ✅ 개별 하드론 상호작용 추적불가 | ✅ 상위 경계 | ✅ OC-1 계산불능 |
| P_open | ✅ 고립 핵 서술 무의미 | ✅ 인터페이스 확보 | ✅ 전자기 장거리 상호작용 |
| P_share | ✅ 개별 원자 전자 추적불가 | ✅ 분자 경계 | ✅ 겹침/혼합 |
| P_stat | ✅ SC-9 개별 서술 불가 | ✅ 분포만 유효 | ✅ 리우빌/에르고딕 |

### 엔트로피 관점

```
S_ext(Π_{k+1}) < S_ext(Π_k)

- 내부 상태 수: 그대로거나 증가
- 외부에서 구분 가능한 상태 수: 감소
- 구분이 줄어들수록 외부 판정은 흔들릴 수가 없다
⇒ 안정성
```

### 핵심 명제

> **"포기(투영)는 정보 손실이 아니라 '쓸 수 없는 정보의 봉인'이다."**

> **"자연의 계층은 새로운 자유도를 더해서 생긴 게 아니라,
> 매 단계마다 '침묵해야 할 자유도'를 하나씩 늘려서 생겼다."**

## 부록 C: 핵심 명제 색인

### 물질의 정의

> **"물질 = 내부 자유도는 더 이상 판정되지 않지만,
> 외부 전이에는 안정적으로 응답하는 경계 상태"**

> **"물질은 '최종 실체'가 아니라
> '설명과 분해가 멈추도록 강제된 경계'다."**

### 표식(Marker)

> **Marker(표식)란
> 외부 관측에서 반복적으로 동일하게 식별되지만,
> 내부 자유도는 더 이상 판정되지 않는 사건-기록의 안정 패턴을 말한다.**

### 계층과 포기

> **"각 단계는 '더 쌓는 과정'이 아니라
> '말할 수 있는 것을 하나씩 버리는 과정'이다."**

> **"자연은 복잡해질수록 더 많은 것을 말할 수 있게 되는 게 아니라,
> 더 많은 것을 *말하지 않기로* 선택한다."**

### 쿼크

> **"쿼크는 '더 이상 쪼갤 수 없어서' 기본이 아니라,
> '이보다 더 내려가면 입자 개념이 붕괴해서' 기본이다."**

### 하드론

> **"하드론은 쿼크들이 '모여서 생긴 우연한 것'이 아니라,
> '모이지 않으면 아무것도 될 수 없어서 생긴 구조'다."**

> **"하드론의 필연성은 QCD의 특정 세부(증명 여부)에 의존하지 않고,
> '부분 자유도를 단독 서술 단위로 인정하지 않는 구조'에서는
> 관측 단위가 조합(경계)으로 수렴한다는 설계적 귀결이다."**

### 분자

> **"원자는 '혼자 있을 수 있는 최소 단위'가 아니라,
> '모이면 반드시 재구성되는 열린 단위'다."**

### 법칙

> **"화학 법칙은 자연이 새로 만든 규칙이 아니라,
> 분자라는 경계가 '불안정한 경우를 다 지워버린 뒤 남긴 흔적'이다."**

### 통계 ★ v1.2 추가

> **"통계 서술은 무지를 보완하는 도구가 아니라,
> SC-9에 의해 살아남은 마지막 서술 형식이다."**

> **"큰 수의 법칙은 평균을 '선택'하게 만드는 규칙이 아니라,
> 개별 서술을 구조적으로 제거하는 강제 메커니즘이다."**

> **"'얼마나 많이 있느냐'는 새로운 개념이 아니라,
> 개별 설명이 무너질 때 자동으로 남는 마지막 좌표다."**

### 투영

> **"포기(투영)는 정보 손실이 아니라
> '쓸 수 없는 정보의 봉인'이다."**

## 부록 D: D설계도가 결정하지 않는 것

### 명시적 비결정 영역

❌ G = SU(3)인 이유 (왜 색이 3개인가)  
❌ 쿼크가 6종인 이유 (세대 수)  
❌ 구체적 전하값 (2/3, -1/3)  
❌ 질량값 (MeV 단위)  
❌ 결합상수 값  
❌ 렙톤이 왜 쿼크와 다른 위상을 가지는가  
   → 렙톤은 MAT-3의 "복합 물질"이 아니라 MAT-2의 "기본 표식" 수준  
❌ 세대가 왜 3개인가

### 해석

이것들은 "아직 모르는 것"이 아니라,
**D설계도 구조상 결정될 수 없는 것** (SC-9: 서술 불완전성)
